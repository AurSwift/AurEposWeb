# Environment Variables Summary

## Required Variables (5)

1. **DATABASE_URL** - PostgreSQL connection string
2. **NEXTAUTH_URL** - Application base URL  
3. **NEXTAUTH_SECRET** - JWT signing secret (generate with: openssl rand -base64 32)
4. **RESEND_API_KEY** - Resend API key for emails
5. **RESEND_FROM_EMAIL** - Email sender (optional, defaults to onboarding@resend.dev)

## Quick Start

1. Create .env.local file
2. Copy template from ENVIRONMENT_SETUP.md
3. Fill in your values
4. Restart dev server

See ENVIRONMENT_SETUP.md for detailed instructions.
